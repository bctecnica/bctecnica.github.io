I’m an aspiring software engineer with a passion for technology, along my journey I fell in love with Front-end Web Development where I am confident in using HTML, CSS and Javascript to help me create elegant and responsive websites. I have been a self taught developer for over a year now and spend whatever spare time I have between work and family life pushing to improve my skills as a developer. I also have a strong interest in both UX and UI design and everything to do with improving the user experience.

I originally started my coding journey building android apps using XML and Java but soon moved into front end web development which I fell in love with. Although I learn and grow more as a developer everyday I feel I’m now confident enough to start looking at web development as a full time career. I am excited by the prospect of joining a team where I can both learn and improve upon my existing skills whilst contributing to larger projects in an ever evolving industry. Lots of my projects are things that were inspired by my friends and family and I am proud of what I have achieved so far so please check out my skills and projects below.
